# Kareem Singleton | AI Implementation Portfolio

Standalone Next.js portfolio for Vercel.

## Local Commands

```bash
npm install
npm run dev
npm run typecheck
npm run build
```

## Vercel Settings

- Framework preset: `Next.js`
- Build command: `npm run build`
- Install command: `npm install`
- Output directory: leave default
- Environment variables: none required for the current static portfolio

## Before Public Launch

- Add final screenshots or short demo captures for the strongest four case studies.
- Update `metadataBase` in `app/layout.tsx` after assigning the final Vercel URL or custom domain.
